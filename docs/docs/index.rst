Sparts' Documentation!
=======================================

**Sparts** allows you to track open source components used in your software supply chain.
Keep track of an Open Source Bill of Matierals, cyrptography data, source code, legal notices, and other Compliance Artifacts.
Before you ship production code, know that your suppliers have provided you with the proper Compliance Artifacts.
Sparts is a decentralized immutable ledger built using Hyperledger Sawtooth.
